# Scripts Directory

This directory contains Python scripts for AI-powered features.

## Files

- `quiz_generator.py` - AI-powered quiz generation from PDF/text content using Mistral API
- `test_quiz.py` - Test script for quiz generation
- `requirements.txt` - Python dependencies

## Setup

```bash
pip install -r requirements.txt
```

## Usage

The quiz generator is primarily used by the backend API. See the main README.md for more information.

## Environment Variables

The quiz generator requires the following environment variable:
- `MISTRAL_API_KEY` - Your Mistral AI API key
